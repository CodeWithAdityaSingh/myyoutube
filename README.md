# YouTube Clone

This project is a YouTube clone built using React, demonstrating a variety of advanced features. It was created with [Create React App](https://github.com/facebook/create-react-app) and showcases several functionalities to mimic YouTube's core features.

## Key Features

- Redux for State Management**: Utilizes Redux to manage and centralize the application's state, ensuring smooth and predictable interactions across the app.
- Advanced Routing**: Implements sophisticated routing techniques to handle multiple views and navigation seamlessly.
- Fetching Multiple APIs: Integrates with various APIs to dynamically fetch and display data, simulating real-world content interactions.
- Auto-Suggestions**: Provides auto-suggestions to enhance the search experience and help users quickly find relevant content.
- N-Level Comment Section**: Features a nested comment system allowing users to engage in conversations with multiple levels of replies.
- Live Chat**: Includes a live chat option to facilitate real-time communication and interaction within the app.

## Getting Started

To get a local copy up and running follow these simple steps:

### Prerequisites

- Node.js (version >= 14.0.0)
- npm (version >= 6.0.0) or yarn

### Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/CodeWithAdityaSingh/myyoutube.git
    ```
2. Navigate into the project directory:
    ```bash
    cd myyoutube
    ```
3. Install the dependencies:
    ```bash
    npm install
    ```

### Running the App

To start the development server:
```bash
npm start
